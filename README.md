VEHICLE RENTAL SYSTEM
A console-based Vehicle Rental System developed using C++ and Object-Oriented Programming concepts.
FEATURES
- Add Cars
- Add Bikes
- Add Customers
- Rent Vehicles
- Return Vehicles
- Show Available Vehicles
CONCEPTS USED
- Inheritance
- Polymorphism
- Virtual Functions
- STL Vector
- DEVELOPED USING
- C++
- VS Code
